Чтобы получить часть строки в PHP используйте функцию substr. Она вырезает и возвращает подстроку из строки. Сама строка при этом не изменяется.

Первым параметром функция получает строку, вторым - номер символа, с которого начинать вырезание (нумерация с нуля), а третьим - количество символов, которое следует вырезать:

Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'bcd' <-php-> <-show->
