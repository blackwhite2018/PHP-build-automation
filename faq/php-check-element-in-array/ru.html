Для проверки элемента массива используется функция in_array.

Пусть дан следующий массив:

<~show~> <+php+> <-php->

<-show->

Давайте проверим с помощью in_array, есть в этом массиве элемент с текстом 'html' или нет:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> true <-php-> <-show->
