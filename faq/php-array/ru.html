Массив в PHP создается с помощью квадратных скобок [ ]. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> ['php', 'html', 'css'] <-php-> <-show->

Ранее для создания массива использовалась функция array, но в настоящее время она не пользуется особой популярностью, хотя работает и ее можно встретить в старом коде.
