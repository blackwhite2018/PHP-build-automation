Первый элемент массива имеет номер 0. Для его получения к нему следует обратиться по этому номеру. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 1 <-php-> <-show->

Можно также использовать функции array_shift. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'a' <-php-> <-show->
