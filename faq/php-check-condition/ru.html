В PHP для таких вопросов предназначена конструкция if, которая позволяет выполнять определенный код при выполнении какого - либо условия. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'Верно!' <-php-> <-show->
