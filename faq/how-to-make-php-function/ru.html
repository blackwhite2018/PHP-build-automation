В PHP функция делается с помощью команды function. Далее через пробел следует имя функции и круглые скобки.

Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Вместо команды echo обычно используется return. В этом случае результат работы функции не выводится сразу на экран, а может быть записан в переменную.

Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Круглые скобки могут быть пустыми, либо в них могут быть указаны параметры, которые принимает функция. Параметры - это обычные переменные PHP.

Давайте сделаем функцию, которая параметром будет принимать число, а возвращать его квадрат:

<~show~> <+php+> <-php-> <-show->
