Самый просто способ добавления элемента в массив

Самый простой и очевидный способ - вставить в массива новый элемент. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> [1, 2, 3, 4, 5] <-php-> <-show->
