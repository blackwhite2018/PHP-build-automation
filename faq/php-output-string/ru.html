{{title: PHP вывод строки}} {{h1: ~ }} {{meta: Строка является одним из типов данных в PHP. Нажмите, чтобы посмотреть подробности и примеры работы.}}

Одним из простых типов данных в PHP являются строки. Строки требуется брать в кавычки (одинарные или двойные - без разницы). Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'word' <-php-> <-show->
