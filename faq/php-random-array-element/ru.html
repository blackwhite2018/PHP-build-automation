В PHP случайный элемент массива можно вывести с помощью функции array_rand. Эта функция возвращает случайный ключ, а не элемент, но зная ключ мы можем получить и значение элемента. Смотрите пример:

<~show~> <+php+> 1, 'b'=>2, 'c'=>3, 'd'=>4, 'e'=>5]; $key = array_rand($arr); var_dump($key); //случайный ключ var_dump($arr[$key]); //случайный элемент ?> <-php-> <-show->
