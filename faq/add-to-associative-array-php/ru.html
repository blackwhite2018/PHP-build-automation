{{title: Добавить в ассоциативный массив PHP}} {{h1: ~ }} {{meta: На данной странице показаны способы добавления элемента в ассоциативный массив PHP. Нажмите, чтобы посмотреть подробности.}}

Пусть у нас есть следующий ассоциативный массив:

<~show~> <+php+> 1, 'b' => 2, 'c' => 3 ]; ?> <-php-> <-show->

Давайте добавим в него элемент с ключом 'd' и значением 4:

<~show~> <+php+> 1, 'b' => 2, 'c' => 3 ]; $arr['d'] = 4; // добавим новый элемент в массив var_dump($arr); ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> ['a' => 1, 'b' => 2, 'c' => 3, 'd' => 4] <-php-> <-show->
