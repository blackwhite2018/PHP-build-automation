Получения значения из массива

Значение массива в PHP мы получаем по обращению к ключу (для получения подробной информации перейтиде сюда). Смотрите пример:

Ключом является цифра

<~show~> <+php+> 'пн', 2=>'вт', 3=>'ср', 4=>'чт', 5=>'пт', 6=>'сб', 7=>'вс']; echo $arr[1]; ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'пн' <-php-> <-show->

Ключом является строка

<~show~> <+php+> 1, 'вт'=>2, 'ср'=>3, 'чт'=>4, 'пт'=>5, 'сб'=>6, 'вс'=>7]; echo $arr['пн']; ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 1 <-php-> <-show->

Получение всех значений из массива с помощью функции array_values

Для получения всех значений из массива в PHP вы можете использовать функцию array_values. Смотрите пример:

<~show~> <+php+> 'пн', 2=>'вт', 3=>'ср', 4=>'чт', 5=>'пт', 6=>'сб', 7=>'вс']; var_dump(array_values($arr)); ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> ['пн', 'вт', 'ср', 'чт', 'пт', 'сб', 'вс'] <-php-> <-show->
