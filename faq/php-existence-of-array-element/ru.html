{{title: PHP существование элемента массива}} {{h1: ~ }} {{meta: Проверки элемента массива на существование используется функция in_array. Нажмите, чтобы посмотреть подробности и примеры работы.}}

Функция in_array проверяет наличие заданного элемента в массиве. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> true <-php-> <-show->
