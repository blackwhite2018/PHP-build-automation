{{title: PHP поменять местами элементы массива}} {{h1: ~ }} {{meta: Для работы с заменой позиций в массиве есть функция array_flip. Нажмите, чтобы посмотреть подробности и примеры работы.}}

Функция array_flip производит обмен местами ключей и значений массива. Смотрите пример:

<~show~> <+php+> 1, 'b'=>2, 'c'=>3]; $a = array_flip($arr); var_dump($a); ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> [1=>'a', 2=>'b', 3=>'c'] <-php-> <-show->
