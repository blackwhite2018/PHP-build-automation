Массив в PHP создается с помощью [] ( до версии PHP 5. 4 массив создавался с помощью функции []). Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> ['php', 'html', 'css'] <-php-> <-show->
