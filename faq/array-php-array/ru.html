В настоящее время более современным вариантом является создание массива с помощью квадратных скобок [ ], вот так:

<~show~> <+php+> <-php-> <-show->

Обратиться к какому - нибудь можно по его номеру (нумерация с нуля). Выведем, например, элемент 'a':

<~show~> <+php+> <-php-> <-show->

А теперь элемент 'b':

<~show~> <+php+> <-php-> <-show->

Можно делать массивы со своими ключами. Такие массивы называются ассоциативными. Давайте сделаем такой массив:

<~show~> <+php+> 1, 'key2'=>2, 'key3'=>3]; ?> <-php-> <-show->

Давайте выведем, например, элемент с ключом 'key1':

<~show~> <+php+> 1, 'key2'=>2, 'key3'=>3]; echo $arr['key1']; //выведет 1 ?> <-php-> <-show->

А теперь с ключом 'key2':

<~show~> <+php+> 1, 'key2'=>2, 'key3'=>3]; echo $arr['key2']; //выведет 2 ?> <-php-> <-show->
