{{title: PHP поиск в строке}} {{h1: ~ }} {{meta: В PHP есть специальная функция strpbrk для поиска в строке. Нажмите, чтобы посмотреть подробности и примеры работы.}}

В PHP функция strpbrk выполняет поиск в строке любого заданного символа. Возвращает строку, начиная с найденного символа, или false, если он не был найден. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'is very big' <-php-> <-show->
