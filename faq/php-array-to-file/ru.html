Это одна из вариаций добавления массива в txt файл. Для этого нам нужны функции fopen и fwrite. Смотрите пример:

<~show~> <+php+> 'val', 'key2' => 'val2', ['key3' => 'val3']]; $file = fopen('file. txt','w'); fwrite($file, print_r($arr, 1)); fclose($file); ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> '['key' => 'val', 'key2' => 'val2', ['key3' => 'val3']]' <-php-> <-show->
