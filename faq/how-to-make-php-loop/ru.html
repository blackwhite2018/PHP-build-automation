С помощью цикла можно выполнить некоторый кусочек кода заданное количество раз. В PHP есть три цикла: while, for и foreach. Давайте посмотрим, как их сделать.

Цикл while

Цикл while будет выполняться до тех пор, пока верно выражение, переданное ему параметром. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 12345 <-php-> <-show->

Цикл for

Цикл for похож while, только более компакнтый. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 12345 <-php-> <-show->

Цикл foreach

Цикл foreach используется для прохождения по всем элементам массива. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 12345 <-php-> <-show->
