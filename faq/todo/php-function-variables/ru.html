В PHP функция задается с помощью команды function. Далее через пробел следует имя функции и круглые скобки. Круглые скобки могут быть пустыми, либо в них могут быть указаны параметры, которые принимает функция. Параметры - это обычные переменные PHP. Смотрите пример:

<+php+> <-php->

Результат выполнения кода:

<+php+> 'Привет, мир!' <-php->
