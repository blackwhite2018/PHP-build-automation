Цикл со счетчиком используется для выполнения тела цикла определенное число раз. С помощью цикла for можно (и нужно) создавать конструкции, которые будут выполнять действия совсем не такие тривиальные, как простая переборка значения счетчика. Смотрите пример:

<+php+> 10) { break; } echo $i; } ?> <-php->

Результат выполнения кода:

<+php+> 12345678910 <-php->
