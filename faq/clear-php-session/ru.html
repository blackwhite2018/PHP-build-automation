{{title: Очистить сессию PHP}} {{h1: ~ }} {{meta: Полностью очистить сессию можно с помощью функции session_destroy. Можно также удалить конкретную переменную сессии.}}

Полностью очистить сессию можно с помощью функции session_destroy. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Конкретную переменную сессии можно удалить с помощью функции unset. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Учтите, что для очистки сессии ее сначала нужно запустить через session_start.
