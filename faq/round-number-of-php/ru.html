В PHP для округления чисел используется функция round. Смотрите пример округления до целых:

<~show~> <+php+> <-php-> <-show->

Вторым параметром функции можно указать количество знаков в дробной части. Давайте округлим до десятых и до сотых:

<~show~> <+php+> <-php-> <-show->

Можно также округлить число в верхнюю сторону. Для этого используется функция ceil. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Можно также округлить число в нижнюю сторону. Для этого используется функция floor. Смотрите пример:

<~show~> <+php+> <-php-> <-show->
