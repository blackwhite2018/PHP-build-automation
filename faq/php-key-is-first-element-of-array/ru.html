{{title: PHP ключ первый элемент массива}} {{h1: ~ }} {{meta: Для получения первого элемента массива в PHP есть функция array_shift. Нажмите, чтобы посмотреть подробности и примеры работы.}}

Так как в программировании нумерация с нуля, то ключ первого элемента массива - это 0. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 1 <-php-> <-show->

Для вывода первого элемента массива используется функция array_shift. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'a' <-php-> <-show->
