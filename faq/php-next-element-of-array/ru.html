{{title: PHP следующий элемент массива}} {{h1: ~ }} {{meta: В PHP есть функция котороая проходит по массиву. Нажмите, чтобы посмотреть подробности и примеры работы.}}

Для переключение между элементами массива в PHP есть функция next. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'html' <-php-> <-show->
