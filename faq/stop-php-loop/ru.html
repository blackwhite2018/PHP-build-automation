Для этого, чтобы остановить цикл в PHP, следует воспользоваться оператором break.

В следующем примере мы перебираем массив с помощью цикла foreach и, как только нам встретится число 3 - выходим из цикла:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 123 <-php-> <-show->

Оператор break работает также в циклах for и while. Используете его по аналогии с предыдущем примером.
