Сортировка массива с использованием sort

Функция sort используется для сортировки массива в алфавитном порядке. Смотрите пример:

<~show~> <+php+> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> ['ананас', 'банан', 'лимон', 'яблоко'] <-php-> <-show->

Сортировка массива с использованием ksort

Функция ksort используется для сортировки массива по ключу. Смотрите пример:

<~show~> <+php+> лимон', 'д'=>'банан', 'a'=>'яблоко', 'в'=>'ананас']; ksort($arr); var_dump($arr); ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> ['а'=>'яблоко', 'б'=> лимон', 'в'=>'ананас', 'д'=>'банан'] <-php-> <-show->

Сортировка массива с использованием asort

Функция asort используется для сортировки массива по значению с сохранением ключей. Смотрите пример:

<~show~> <+php+> лимон', 'д'=>'банан', 'a'=>'яблоко', 'в'=>'ананас']; asort($arr); var_dump($arr); ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> ['в'=>'ананас', 'д'=>'банан', 'б'=> лимон', а'=>'яблоко'] <-php-> <-show->
