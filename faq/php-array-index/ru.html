Получения значения из массива

В PHP иднексами массива является ключи массива (для получения подробной информации перейтиде сюда). Смотрите пример:

Получения значения из массива. Ключом является цифра

<~show~> <+php+> 'пн', 2=>'вт', 3=>'ср', 4=>'чт', 5=>'пт', 6=>'сб', 7=>'вс']; echo $arr[1]; ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 'пн' <-php-> <-show->

Получения значения из массива. Ключом является строка

<~show~> <+php+> 1, 'вт'=>2, 'ср'=>3, 'чт'=>4, 'пт'=>5, 'сб'=>6, 'вс'=>7]; echo $arr['пн']; ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> 1 <-php-> <-show->

Получение всех ключей из массива с помощью функции array_keys

Для получения всех ключей из массива в PHP вы можете использовать функцию array_keys. Смотрите пример:

<~show~> <+php+> 'пн', 2=>'вт', 3=>'ср', 4=>'чт', 5=>'пт', 6=>'сб', 7=>'вс']; var_dump(array_keys($arr)); ?> <-php-> <-show->

Результат выполнения кода:

<~show~> <+php+> [ 1, 2, 3, 4, 5, 6, 7] <-php-> <-show->
