Массивы

В JavaScript массивы создается при помощи квадратных скобок или ключевого слова new Array:

<~show~> <+javascript+> let arr1 = ['a', 'b', 'c', 'd', 'e']; let arr2 = new Array(1, 2, 3); <-javascript-> <-show->

Оба массива практически идентичны. К элементам этих массив можно обращаться по ключу. То есть, первый элемент массива имеет ключ 0, второй 1 и так далее. Смотрите пример:

Давайте выведем 'c':

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; console.log(arr[2]); // выведет 'c' <-javascript-> <-show->
