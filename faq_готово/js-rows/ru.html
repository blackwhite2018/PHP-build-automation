В JavaScript есть не только числовой тип данных, но и строковый.

Строки создаются при помощи кавычек:

<~show~> <+javascript+> let str = 'abc'; console.log(str); // выведет 'abc' <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> 'abc' <-javascript-> <-show->

Сложение строк

Со строками можно производить операцию конкатенации (сложения):

<~show~> <+javascript+> let str1 = 'abc'; let str2 = 'de'; console.log(str1 + str2 + '!'); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> 'abcde!' <-javascript-> <-show->

Длина строк

Длину строки можно узнать при помощи метода length:

<~show~> <+javascript+> let str = 'abcde'; console.log(str,length); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> 5 <-javascript-> <-show->
