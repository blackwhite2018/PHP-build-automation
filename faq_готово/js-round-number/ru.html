Округлить до ближайшего целого

Чтобы округлить число до ближайшего целого по правилам математического округления, используйте Math.round. Смотрите пример:

<~show~> <+javascript+> console.log(Math.round(3.4)); // выведет 3 console.log(Math.round(3.5)); // выведет 4 console.log(Math.round(3.6)); // выведет 4 <-javascript-> <-show->

Округлить до указанного количества знаков в дробной части

Чтобы округлить число с заданным количеством знаком в дробной части, используйте toFixed. Смотрите пример:

<~show~> <+javascript+> let num = 3.1257; console.log(num.toFixed(1)); // выведет 3.1 console.log(num.toFixed(2)); // выведет 3.13 console.log(num.toFixed(3)); // выведет 3.126 <-javascript-> <-show->

Округлить в большую сторону

Чтобы округлить в большую сторону, используйте Math.ceil. Смотрите пример:

<~show~> <+javascript+> console.log(Math.ceil(3.1)); // выведет 4 console.log(Math.ceil(3.5)); // выведет 4 console.log(Math.ceil(3.6)); // выведет 4 <-javascript-> <-show->

Округлить в меньшую сторону

Чтобы округлить в меньшую сторону, используйте Math.floor. Смотрите пример:

<~show~> <+javascript+> console.log(Math.floor(3.4)); // выведет 3 console.log(Math.floor(3.5)); // выведет 3 console.log(Math.floor(3.9)); // выведет 3 <-javascript-> <-show->
