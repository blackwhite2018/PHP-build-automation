Оператор instanceof

Оператор instanceof позволяет проверить, какому классу принадлежит объект, с учетом прототипного наследования. Смотрите пример:

<~show~> <+javascript+> console.log([] instanceof Array); // выведет true <-javascript-> <-show->

Метод Array.isArray

Метод Array.isArray возвращает true, если объект является массивом, иначе false. Смотрите пример:

<~show~> <+javascript+> console.log(Array.isArray([])); // выведет true <-javascript-> <-show->
