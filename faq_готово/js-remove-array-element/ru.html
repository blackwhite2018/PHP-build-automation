Удаление элемента массива

Давайте зададим массив, с которым посмотрим способы удаления элементов массива: Массивы являются объектами, для удаления ключа можно воспользоваться обычным delete. Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; <-javascript-> <-show->

Оператор delete

В JavaScript массивы являются объектами и обладают ключами, давайте удалим элемент массива по ключу при помощи оператора delete:

<~show~> <+javascript+> delete arr[1]; console.log(arr[1]); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> undefined <-javascript-> <-show->

Метод splice

Также удалить элементы массива можно при помощи метода splice

<~show~> <+javascript+> arr.splice(1, 1); console.log(arr); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> ['a', 'c', 'd', 'e'] <-javascript-> <-show->
