Добавить элемент массива через квадратные скобки

В любой массив можно добавлять элементы с новым ключом. Смотрите пример:

<~show~> <+javascript+> let arr = []; arr[0] = 'a'; arr[1] = 'b'; arr[2] = 'c'; console.log(arr); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> ['a', 'b', 'c'] <-javascript-> <-show->

Метод push

Метод push добавляет неограниченное количество элементов в конец массива. Элементы передаются параметром метода. Метод изменяет исходный массив, возвращает новую длину массива. Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c']; arr.push('d', 'e'); console.log(arr); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> ['a', 'b', 'c', 'd', 'e'] <-javascript-> <-show->

Метода unshift

Метод unshift добавляет любое количество новых элементов в начало массива. Метод изменяет исходный массив и возвращает новую длину массива. Смотрите пример:

<~show~> <+javascript+> let arr = ['c', 'd', 'e']; arr.unshift('a', 'b'); console.log(arr); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> ['a', 'b', 'c', 'd', 'e'] <-javascript-> <-show->
