Конкатенация строк через знак '+'

Для конкатенации строк в JavaScript используется оператор '+'. Смотрите пример:

<~show~> <+javascript+> let str = 'a' + 'b'; console.log(str); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'ab' <-javascript-> <-show->

Метод concat

Строки также можно соединять через метод concat. Смотрите пример:

<~show~> <+javascript+> let str = 'a'; let result = str.concat('b'); console.log(result); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'ab' <-javascript-> <-show->
