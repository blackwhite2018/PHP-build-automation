Для работы с датой в JavaScript используется объект new Date. Давайте создадим его:

<~show~> <+javascript+> let date = new Date(); <-javascript-> <-show->

У объекта Date есть методы для получения текущих: месяца, дня и года. Используя методы объекта Date выведем дату в формате 'день.месяц.год':

<~show~> <+javascript+> let result = `${date.getDate()}.${date.getMonth()}.${date.getFullYear()}`; console.log(result); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> '6.2.2020' <-javascript-> <-show->

Стоит учесть, что месяц, который возвращает функция getMonth, начинается с 0.

Исправим код:

<~show~> <+javascript+> let result = `${date.getDate()}.${date.getMonth() + 1}.${date.getFullYear()}`; console.log(result); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> '6.3.2020' <-javascript-> <-show->
