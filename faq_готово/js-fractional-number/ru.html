JavaScript поддерживает так называется 'дробные числа', они же числа с 'плавающей запятой'. Зададим дробное число:

<~show~> <+javascript+> let num = 5.2; console.log(num); // выведет 5.2 <-javascript-> <-show->

В JavaScript также есть функция parseFloat, которая преобразует строку к числу с плавающей запятой:

<~show~> <+javascript+> let str = '5.2'; let float = parseFloat(str); console.log(typeof float, float); // выведет number 5.2 <-javascript-> <-show->
