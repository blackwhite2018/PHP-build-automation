Для перевода массива в строку используйте метод join. Он объединяет элементы массива с указанным разделителем. Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c']; let str = arr.join('-'); console.log(str); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'a-b-c' <-javascript-> <-show->
