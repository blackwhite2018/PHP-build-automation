Достаточно распространенной проблемой являются бесконечные циклы. Если в теле цикла не будет выполнено условие для выхода из него, цикл может выполняться бесконечно. Посмотрите пример:

<~show~> <+javascript+> let bool = false; while (bool) { console.log('Такой цикл будет выполняться бесконечно.'); } <-javascript-> <-show->
