Аналог ассоциативных массивов

В JavaScript нет ассоциативных массивов, но есть объекты, которые позволяют работать с парами 'ключ' => 'значение'. Давайте объявим объект:

<~show~> <+javascript+> let obj = {key1:200, key2:300, key2:400}; <-javascript-> <-show->

Работа с полями объекта как с ключами массива

К объекту можно обращаться по ключу:

<~show~> <+javascript+> console.log(obj[key1]); // выведет 200 <-javascript-> <-show->

Работа с полями объекта как со свойствами

К объекту можно обращаться как к свойству:

<~show~> <+javascript+> console.log(obj.key1); // выведет 200 <-javascript-> <-show->
