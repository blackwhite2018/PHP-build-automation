Вывод массива можно произвести при помощи команды console.log, после чего массив будет доступен для просмотра в консоле инспектора кода.

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; console.log(arr); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> ['a', 'b', 'c', 'd', 'e'] <-javascript-> <-show->
