В JS массивы создаются с помощью квадратных скобок [ ]. Давайте, создадим массив:

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; <-javascript-> <-show->

Выведем c помощью этого массива какой-нибудь элемент. Для этого нужно написать имя массива и квадратные скобки с номером элемента (этот номер называется ключом).

Давайте выведем 'c':

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; console.log(arr[2]); <-javascript-> <-show->
