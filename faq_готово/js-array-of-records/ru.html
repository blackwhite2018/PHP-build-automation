Для организации коллекции записей в JavaScript можно использовать массивы, которые будут хранить объекты. Такие массивы можно перебирать стандартными способами JavaScript и искать нужные записи. Посмотрим пример:

<~show~> <+javascript+> let obj = [ { header: 'запись1', text: 'Lorem ipsum dolor sit amet.' }, { header: 'запись2', text: 'Lorem ipsum dolor sit amet, consectetur adipisicing.' } ]; for (const elem of obj) { console.log(elem.header, elem. text); } <-javascript-> <-show->

Давайте попробуем перебрать записи при помощи цикла for-of и вывести их:

<~show~> <+javascript+> for (const elem of obj) { console.log(elem.header, elem. text); } <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> запись1 Lorem ipsum dolor sit amet. запись2 Lorem ipsum dolor sit amet, consectetur adipisicing. <-javascript-> <-show->
