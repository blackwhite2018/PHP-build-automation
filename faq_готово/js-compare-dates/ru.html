В JavaScript можно сравнивать объекты с датами.

Посмотрим на примере. Пусть даны два объекта с датами, которые мы создали через new Date и записали в переменные. Давайте сравним, какая дата больше:

<~show~> <+javascript+> let date1 = new Date(2020, 1, 1); let date2 = new Date(2021, 2, 3); console.log(date1 < date2); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> true <-javascript-> <-show->
