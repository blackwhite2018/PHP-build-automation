Метод join объединяет элементы массива в строку с указанным разделителем (он будет вставлен между элементами массива). Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; let str = arr.join('-'); console.log(str); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'a-b-c-d-e' <-javascript-> <-show->
