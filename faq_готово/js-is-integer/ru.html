В JavaScript есть 2 типа чисел: целые и с плавающей запятой. Давайте посмотрим, как задавать целые числа:

<~show~> <+javascript+> let num = 5; console.log(num); // выведет 5 <-javascript-> <-show->

С целыми числами можно производить различные арифметические операции. Смотрите пример:

<~show~> <+javascript+> console.log(1 + 1); // выведет 2 console.log(1 - 1); // выведет 0 console.log(1 * 1); // выведет 1 console.log(1 / 1); // выведет 1 <-javascript-> <-show->
