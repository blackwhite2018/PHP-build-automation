Для удаления пробелов можно воспользоваться методом replace. Смотрите пример:

<~show~> <+javascript+> let str = 'а b c'; console.log(str.replace(/ /g, '')); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'abc' <-javascript-> <-show->
