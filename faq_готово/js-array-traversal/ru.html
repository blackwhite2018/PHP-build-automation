Цикл for

В JavaScript обойти элементы массива можно при помощи цикла for. Пусть у нас дан такой массив:

<~show~> <+javascript+> let arr = [1, 2, 3, 4, 5]; <-javascript-> <-show->

Давайте переберём элементы массива при помощи цикла for:

<~show~> <+javascript+> for (let i = 0; i < arr.length; i++) { console.log(arr[i]); } <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->

Цикл for-of

У цикла for в JavaScript есть и другая запись, это цикл for-of. Давайте переберём массив при помощи цикла for-of:

<~show~> <+javascript+> for (const elem of arr) { console.log(elem); } <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->

Метод forEach

В ES5 появились новые методы для работы с массивами, одним из которых является forEach для перебора элементов массива. В качестве аргумента forEach принимает функцию, которая приминяется в каждому элементу массива. Также, эта функция принимает аргументом каждый элемент массива по очереди:

<~show~> <+javascript+> arr.forEach((elem) => { console.log(elem); }); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->
