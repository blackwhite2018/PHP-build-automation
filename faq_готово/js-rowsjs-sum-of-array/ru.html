Давайте напишем функцию, которая параметром будет принимать массив и возвращать сумму его элементов. Для этого создадим функцию getSumElems:

<~show~> <+javascript+> const getSumElems = (arr) => {}; // создаем стрелочную функцию getSumElems <-javascript-> <-show->

Следующим шагом воспользуемся методом reduce, который сворачивает массив к одному значению:

<~show~> <+javascript+> const getSumElems = (arr) => { return arr.reduce((sum, elem) => { // перебираем массив при помощи метода reduce sum = sum + elem; // в промежуточную переменную sum прибавляем значение элемента return sum; // возвращаем промежуточное значение sum }, 0); }; let arr = [2, 1, 3, 4]; console.log(getSumElems(arr)); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> 10 <-javascript-> <-show->

Метод reduce проходит по всем элементам массива, а также в качестве аргумента получает функцию. В нашем случае используется стрелочная функция.

Эта функция получает 2 параметра: промежуточную переменную, в которую можно записывать значения и элемент массива. Следующим шагом мы в промежуточную переменную sum прибавляем значение элемента и возвращаем значение переменной sum.

Если вы не умеете пользоваться методом reduce или не знаете, что означает запись '(arr) =>', тогда изучите возможности ES5/6 по соответствующим ссылкам.

Наша функция работает, но её также можно и улучшить. Если вспомнить каким образом работают стрелочные функции, у нас получается:

<~show~> <+javascript+> const getSumElems = (arr) => arr.reduce((sum, elem) => sum + elem); let arr = [2, 1, 3, 4]; console.log(getSumElems(arr)); <-javascript-> <-show->

Результат работы кода:

<~show~> <+javascript+> 10 <-javascript-> <-show->
