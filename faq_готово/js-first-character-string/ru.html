{{title: JS первый символ строки}} {{h1: ~ }} {{meta: Как получить первый символ строки в JavaScript. Нажмите, чтобы посмотреть подробности и примеры работы.}}

Чтобы получить первый символ строки, обратитесь к нему по его номеру. Первый символ имеет номер ноль. Смотрите пример:

<~show~> <+javascript+> let str = 'abcde'; console.log(str[0])); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'a' <-javascript-> <-show->
