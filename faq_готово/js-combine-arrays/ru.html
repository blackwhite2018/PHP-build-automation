Чтобы объединить массивы в JavaScript, используйте метод concat. Смотрите пример:

<~show~> <+javascript+> let arr1 = ['a', 'b', 'c']; let arr2 = [1, 2, 3]; let arr3 = arr1.concat(arr2); console.log(arr3); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> ['a', 'b', 'c', 1, 2, 3] <-javascript-> <-show->

Также объединить массивы можно при помощи оператора spread. Смотрите пример:

<~show~> <+javascript+> let arr1 = ['a', 'b', 'c']; let arr2 = [1, 2, 3]; let arr3 = [...arr1, ...arr2]; console.log(arr3); <-javascript-> <-show->
