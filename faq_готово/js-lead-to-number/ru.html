В JavaScript есть несколько способов приведения к числу. Самыми распространнёными считаются функция Number и использование знака '+'. Давайте используем оба способа и при помощи оператора typeof посмотрим на тип данных:

<~show~> <+javascript+> let data = false; let num1 = +data; let num2 = Number(data); console.log(typeof num1); // выведет 'number' console.log(typeof num2); // выведет 'number' <-javascript-> <-show->

Функция Number и знак '+' не являются единственными способами приведения к числу. Помимо них, можно использовать функции parseInt и parseFloat.

Функция parseInt

Функция parseInt преобразует строку в целое число.

Эта функция преобразует строку до тех пор, пока это возможно. То есть, строка '5px' будет преобразована в число 5.

Если преобразование невозможно, например, есть строка 'data = 3', функция вернёт NaN.

Посмотрим примеры:

<~show~> <+javascript+> let str1 = '12px'; let str2 = 'f5'; console.log(parseInt(str1)); // выведет 12 console.log(parseInt(str2)); // выведет NaN <-javascript-> <-show->

Функция parseFloat

Функция parseFloat работает аналогично функции parseInt, но преобразует строку в число с плавающей точкой:

<~show~> <+javascript+> let str1 = '12.5px'; let str2 = 'f5.3'; console.log(parseFloat(str1)); // выведет 12.5 console.log(parseFloat(str2)); // выведет NaN <-javascript-> <-show->
