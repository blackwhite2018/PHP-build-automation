Преобразование даты к числу

При помощи функции Number или знака '+' можно преобразовать объект Date в число, количество миллисекунд, прошедшее с 1 января 1970:

<~show~> <+javascript+> const date = new Date(); console.log(+date); // выведит 1583437957606 console.log(Number(date)); // выведит 1583437957606 <-javascript-> <-show->

Разница между датами

Даты, представленные в виде объекта Date, можно вычитать друг из друга, результат их вычитания – разница в миллисекундах. Смотрите пример:

<~show~> <+javascript+> let now = new Date(); let date = new Date(2015, 23, 4, 12, 59, 59); let diff = now - date; console.log(diff); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 6892970177 <-javascript-> <-show->
