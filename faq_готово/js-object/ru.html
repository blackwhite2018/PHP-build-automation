В языке JavaScript нет ассоциативных массивов, но есть их аналог. Под аналогом подразумеваются объекты, у которых также есть ключи и значения.

Объекты создаются при помощью фигурных скобок { }, внутри которых пишутся элементы этого объекта в формате ключ-значение:

<~show~> <+javascript+> let obj = { a: 1, b: 2, c: 3 }; <-javascript-> <-show->

К элементам объекта можно обращаться по ключу или как к свойствам

<~show~> <+javascript+> console.log(obj[a]); // выведет 1 console.log(obj.a); // выведет 1 <-javascript-> <-show->
