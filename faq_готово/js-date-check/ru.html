Давайте напишем функцию, которая проверяет корректность даты в формате 'дд-мм-гггг'. Назовём функцию isValidDate и в качестве аргумента будем ей передавать дату:

<~show~> <+javascript+> const isValidDate = (date) => { // тело функции }; <-javascript-> <-show->

Следующим шагом преобразуем дату из строки в массив при помощи метода split и для более удобной работы деструктуризируем полученный массив:

<~show~> <+javascript+> const isValidDate = (date) => { let [day, month, year] = date.split('-'); }; <-javascript-> <-show->

Теперь, можно добавить проверку на корректность даты:

<~show~> <+javascript+> const isValidDate = (date) => { let [day, month, year] = date.split('-'); return day > 0 && day < 32 && month > 0 && month < 13 && year > 0 && year < 3000; }; console.log(isValidDate('08-06-1998')); // true console.log(isValidDate('08-1998')); // false <-javascript-> <-show->
