{{title: JS первый элемент массива}} {{h1: ~ }} {{meta: Как получить первый элемент массива в JavaScript. Нажмите, чтобы посмотреть подробности и примеры.}}

Чтобы получить первый элемент массива, обратитесь к нему по его номеру. Первый элемент имеет номер ноль. Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c']; console.log(arr[0]); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'a' <-javascript-> <-show->

Можно также воспользоваться методом shift, который извлекает и возвращает первый элемент массива. После извлечения элемент пропадает из массива. Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c']; console.log(arr.shift()); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'a' <-javascript-> <-show->
