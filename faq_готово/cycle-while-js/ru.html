Цикл while будет выполняться до тех пор, пока верно(истинно) выражение, переданное ему параметром. Смотрите пример:

<~show~> <+javascript+> let i = 0; while (i < 5) { i++; console.log(i); } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->
