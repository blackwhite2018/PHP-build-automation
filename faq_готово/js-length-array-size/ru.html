Для того, чтобы найди длину или размер массива в JS используйте свойство length. Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; console.log(arr.length); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 5 <-javascript-> <-show->
