Чтобы найти рандомное число в JavaScript, используйте функцию Math.random. Она возвращает случайное дробное число от 0 до 1. Смотрите пример:

<~show~> <+javascript+> console.log(Math.random()); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 0.765557653267769 <-javascript-> <-show->

Чтобы найти случайное целое число в заданном диапазоне, используйте следующую функцию:

<~show~> <+javascript+> function getRandomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; } <-javascript-> <-show->

Давайте с ее помощью найдем случайное целое число от 10 до 100:

<~show~> <+javascript+> function getRandomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; } console.log(getRandomInt(10, 100)); <-javascript-> <-show->
