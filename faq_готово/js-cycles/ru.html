Цикл while

Цикл while будет выполняться до тех пор, пока верно выражение, переданное ему параметром. Смотрите пример:

<~show~> <+javascript+> let i = 0; while (i < 5) { console.log(i); i++; } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 01234 <-javascript-> <-show->

Цикл for

Цикл for является более компактным вариантом while. Смотрите пример:

<~show~> <+javascript+> for (let i = 1; i <= 5; i++) { console.log(i); } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->

Цикл for также можно использовать для перебора массива:

<~show~> <+javascript+> let arr = [1, 2, 3, 4, 5]; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->

Цикл for-of

Цикл for-of обходит итерируемые объекты (массивы, коллекции set и map, список аргументов и т.д.):

<~show~> <+javascript+> let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { console.log(elem); } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->

Метод forEach

Можно также использовать метод forEach. Смотрите пример:

<~show~> <+javascript+> let arr = [1, 2, 3, 4, 5]; arr.forEach((elem) => { console.log(elem); }); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->

Цикл for-in

Для перебора объекта используется так называемый цикл for-in. Для этого используют такую конструкцию: for (key in obj), где obj - это объект, который мы перебираем, а key - это переменная, в которую последовательно будут ложиться ключи объекта. Смотрите пример:

<~show~> <+javascript+> let obj = {a: 1, b: 2, c: 3}; for (key in obj) { console.log(key); } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> abc <-javascript-> <-show->

Давайте выведем не ключи, а элементы объекта:

<~show~> <+javascript+> let obj = {a: 1, b: 2, c: 3}; for (key in obj) { console.log(obj[key]); } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 123 <-javascript-> <-show->
