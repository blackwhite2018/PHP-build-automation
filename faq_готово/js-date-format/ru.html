Метод Date.parse возвращает количество миллисекунд до даты переданной в параметре в виде строки формата ISO. Если переданная дата будет некорректной, Date.parse вернет NaN. Смотрите пример:

<~show~> <+javascript+> console.log(Date.parse('2017-02-26T13:51:50.417-07:00')); // выведет 1488142310417 console.log(Date.parse('Некорректная дата')); // выведет NaN <-javascript-> <-show->
