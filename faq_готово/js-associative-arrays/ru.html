Ассоциативные массивы в JavaScript делаются с помощью объектов.

Объекты создаются с помощью фигурных скобок { }, внутри которых пишутся элементы этого объекта в формате ключ-значение. Синтаксис здесь такой: ключ, затем идет двоеточие :, а потом значение. Смотрите пример:

<~show~> <+javascript+> let obj = {a: 1, b: 2, c: 3}; console.log(obj['a']); // выведет 1 <-javascript-> <-show->

Есть также альтернативный синтаксис обращения к элементу объекта - через свойство. Смотрите пример:

<~show~> <+javascript+> let obj = {a: 1, b: 2, c: 3}; console.log(obj.a); // выведет 1 <-javascript-> <-show->
