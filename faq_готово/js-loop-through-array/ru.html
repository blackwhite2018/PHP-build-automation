Цикл for

Перебрать массив можно при помощи цикла for. Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; for (let i = 0; i < arr.length; i++) { console.log(arr[i]); } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 12345 <-javascript-> <-show->

Цикл for-of

Цикл for-of обходит итерируемые объекты (массивы, коллекции set иmap, список аргументов и т.д.):

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; for (let elem of arr) { console.log(elem); } <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'abcde' <-javascript-> <-show->

Метод forEach

Можно также для перебора элементов массива использовать метод forEach. Смотрите пример:

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; arr.forEach(function(elem) { console.log(elem); }); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'abcde' <-javascript-> <-show->
