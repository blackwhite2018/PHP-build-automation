Массив представляет собой переменную, в которой в упорядоченном виде можно хранить целый набор какие-то значений. Для создания массива используются квадратные скобки:

<~show~> <+javascript+> let arr = []; // создаем массив arr <-javascript-> <-show->

Пока созданный массив не содержит никаких данных. Заполним его данными:

<~show~> <+javascript+> let arr = [1, 2, 3, 4, 5]; <-javascript-> <-show->

Каждое значение списка, которое мы записали в массив, называется элементом массива.

Элементы разделяются между собой запятой. После этой запятой можно ставить пробелы, а можно и не ставить (для читаемости кода принято ставить).

Кроме чисел, в массиве можно хранить строки. Каждый элемент строки нужно брать в ковычки:

<~show~> <+javascript+> let arr = ['a', 'b', 'c', 'd', 'e']; <-javascript-> <-show->

Кроме строк и чисел в массивах можно хранить и другие типы данных. Кроме того, в одном массиве могут быть разные типы данных ( числа, строки, null, undefined, true, false, infinity):

<~show~> <+javascript+> let arr = ['a', 'b', 4, undefined, true, null]; <-javascript-> <-show->
