Числа и операции над ними в JavaScript

В JavaScript можно выполнять математические операции над числами. Сложение выполняется с помощью оператора +б вычитание с помощью -, умножение с помощью *, а деление с помощью /. Смотрите примеры:

<~show~> <+javascript+> let a = 1 + 2; // выполняем сложение console.log(a); // выведет 3 let b = 3 - 2; // выполняем вычитание console.log(b); // выведет 1 let c = 3 * 2; // выполняем умножение console.log(c); // выведет 6 let d = 4 / 2; // выполняем деление console.log(d); // выведет 2 <-javascript-> <-show->

Операции над переменными

Математические операции можно производить не только над числами, но и над переменными. Сложим, к примеру, значения двух переменных:

<~show~> <+javascript+> let a = 1; let b = 2; console.log(a + b); // выведет 3 <-javascript-> <-show->

Приоритет операций

Математические операции JavaScript имеют такой же приоритет, как в обычной математике. То есть в начале выполняется умножение и деление, а потом уже сложение и вычитание. В следующем примере вначале 2 умножится на 2 и затем к результату прибавится 3:

<~show~> <+javascript+> let a = 2 * 2 + 3; console.log(a); // выведет 7 <-javascript-> <-show->

Равный приоритет

Умножение и деление имеют равный приоритет и выполняются по очереди слева направо. Рассмотрим на примере, что имеется ввиду. В следующем коде вначале выполнится деление, а потом умножение:

<~show~> <+javascript+> let a = 8 / 2 * 4; console.log(a); // выведет 16 <-javascript-> <-show->

Если же переставить знаки местами, то вначале выполнится умножение, а потом деление:

<~show~> <+javascript+> let a = 8 * 2 / 4; console.log(a); // выведет 4 <-javascript-> <-show->

Группирующие скобки

При желании вы можете указывать приоритет операций с помощью группирующих круглых скобок.

Давайте, например, переделаем наш код так, чтобы вначале выполнилось сложение, а уже потом умножение:

<~show~> <+javascript+> let a = 2 * (2 + 3); console.log(a); // выведет 10 <-javascript-> <-show->

Дроби

Ранее мы с вами работали только с целыми числами. Но в JavaScript также существуют и десятичные дроби.

При этом целая и дробная части отделяются друг от друга точкой. Смотрите пример:

<~show~> <+javascript+> let a = 0.5; console.log(a); // выведет 0.5 <-javascript-> <-show->

Отрицательные числа

Числа могут быть отрицательными. Для этого перед числом необходимо поставить знак минус:

<~show~> <+javascript+> let a = -1; console.log(a); // выведет -1 <-javascript-> <-show->
