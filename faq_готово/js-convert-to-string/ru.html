В JavaScript при помощи функции String можно привести различные типы данных к строке. Давайте попробуем на следующем примере:

<~show~> <+javascript+> console.log(String(12)); // '12' console.log(String(null)); // 'null' console.log(String(NaN)); // 'NaN' console.log(String(undefined)); // 'undefined' console.log(String([2, 1, false])); // '2,1,false' console.log(String(false)); // 'false' <-javascript-> <-show->
