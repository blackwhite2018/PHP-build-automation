Метод sort

Метод sort производит сортировку массива в лексикографическом порядке. Также можно указать параметром собственную функцию для сортировки, но это необязательно. Смотрите пример:

<~show~> <+javascript+> let arr = ['c', 'b', 'a', 'e', 'd']; console.log(arr.sort()); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> ['a', 'b', 'c', 'd', 'e'] <-javascript-> <-show->

Cоздание функции сортировки

Метод sort также принимает функцию сравнения вторым параметром. Функция сравнения принимает в качестве аргументов 2 значения. Если первый аргумент меньше второго, функция должна вернуть -1, если первый аргумент больше второго, функция должна вернуть 1, а если аргументы равны, то 0. Давайте посмотрим пример:

<~show~> <+javascript+> const sortArr = (num1, num2) => { if (num1 < num2) { return -1; } if (num1 > num2) { return 1; } return 0; } let arr = ['c', 'b', 'a', 'e', 'd']; console.log(arr.sort(sortArr)); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> ['a', 'b', 'c', 'd', 'e'] <-javascript-> <-show->

Метод sort для чисел

Если требуется отсортировать, например, массив чисел, тогда функцию сравнения можно сократить, вычитая из первого аргумента второй. Давайте посмотрим пример:

<~show~> <+javascript+> const sortArr = (num1, num2) => num1 - num2; let arr = [4, 2, 5, 1, 3]; console.log(arr.sort(sortArr)); <-javascript-> <-show->
