Для замены символов в строке можно использовать метод replace. Давайте, например, с его помощью поменяем букву 'a' на букву 'd':

<~show~> <+javascript+> let str = 'abc'; console.log(str.replace('a', 'd')); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> 'dbc' <-javascript-> <-show->
