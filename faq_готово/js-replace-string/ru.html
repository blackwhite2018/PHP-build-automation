Для замены символов в строке можно воспользоваться методом replace. Смотрите пример:

<~show~> <+javascript+> let str = '1a2b1a'; str.replace(/1/g, '!'); console.log(str); <-javascript-> <-show->

Результат выполнения кода:

<~show~> <+javascript+> '!a2b!a' <-javascript-> <-show->
